Ниже приведена программа, записанная на пяти
языках программирования.



Бейсик

DIM s, t AS INTEGER
 INPUT s
 INPUT t
 IF s > 5 AND t > 5 THEN
    PRINT ‘YES’
 ELSE
    PRINT ‘NO’
 ENDIF
____________________________________________


Python

s = int(input())
t = int(input())
if s > 5 and t > 5:
    print("YES")
else:
    print("NO")
____________________________________________


Паскаль

var s, t: integer;
begin
    readln(s);
    readln(t);
    if (s > 5) and (t > 5)
        then writeln('YES')
        else writeln('NO')
end.

С++

#include <iostream>
using namespace std;
int main() {
    int s, t;
    cin >> s;
    cin >> t;
    if (s > 5 && t > 5)
        cout << "YES";
    else
        cout << "NO";
return 0;
}
