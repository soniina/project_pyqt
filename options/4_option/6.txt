Ниже приведена программа, записанная на пяти
языках программирования.



Бейсик

DIM s, t AS INTEGER
 INPUT s
 INPUT t
 IF s > 9 OR t > 9 THEN
    PRINT ‘YES’
 ELSE
    PRINT ‘NO’
 ENDIF
____________________________________________


Python

s = int(input())
t = int(input())
if s > 9 or t > 9:
    print("YES")
else:
    print("NO")
____________________________________________


Паскаль

var s, t: integer;
begin
    readln(s);
    readln(t);
    if (s > 9) or (t > 9)
        then writeln('YES')
        else writeln('NO')
end.
____________________________________________

С++

#include <iostream>
using namespace std;
int main() {
    int s, t;
    cin >> s;
    cin >> t;
    if (s > 9 || t > 9)
        cout << "YES";
    else
        cout << "NO";
return 0;
}
